hexo命令

Created2020-09-10|Updated2024-02-26

老是记不住 hexo 命令，总结一下。顺便试着用英语来写一下，顺便 blog 无法访问了，目前还搞不清楚什么原因，明明 Github 仓库里都有。

I always forget the hexo commands. I sum up them from this title.

  1. pwd //Return to the location file
  2. cd Ps:(cd D:\\blog) //jump to the file what you input
  3. hexo s //creat the local server to preview theme (local host 4000)
  4. hexo new (Ps: hexo new "hello") //Create a new article named hello
  5. hexo d //hexo deploy
  6. hexo clean //Clear cache file
  7. hexo g //hexo generate
  8. git clone +address //download theme needed

If you finished the blog, input these commands to creat your blog.

  1. hexo clean //清除缓存文件 (db.json) 和已生成的静态文件 (public) clear cache and built public file
  2. hexo g //生成缓存和静态文件 create cache and public file
  3. hexo d //重新部署到服务器 redeploy to server
